Control Tower (React UI)

The visual command center for monitoring workflows, approving gates, and tracking every build in real time.

Control Tower is the React-based web dashboard that sits on top of the ProjectZero API and Temporal. It gives you a live view of every workflow, every stage, every approval, and every failure without touching the CLI.

Service URLs

When running locally, these are the default endpoints:

localhost:3000
Control Tower React UI. The dashboard, workflow detail, and approvals pages live here.
localhost:8000
ProjectZero API. All workflow CRUD, ticket management, and approval endpoints.
localhost:8233
Temporal Web UI. Low-level workflow execution history, task queues, and namespace management.

When to use which

Control Tower (3000) is your daily driver. Use it to check status, approve gates, and drill into failures. The API (8000) is for programmatic access and CLI commands under the hood. Temporal UI (8233) is for deep debugging: if a workflow is stuck, you can inspect the raw event history, signal it, or terminate it from there.
