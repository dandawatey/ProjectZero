Getting Started

Go from zero to a governed AI product in three commands. Install the prerequisites, initialize the factory, bootstrap your first product.

Prerequisites

All five must be present before running /factory-init.

🐳
Docker
Docker Desktop 4+
Runs Postgres and Temporal containers locally.
⬡
Node.js
v18 or later
Powers the React Control Tower and Claude Code CLI.
🐍
Python
3.11 or later
Runs the FastAPI backend and Temporal worker.
◆
Claude Code CLI
latest
The agent runtime. Install via npm — see below.
🔑
ANTHROPIC_API_KEY
set in environment
Required by all Claude agents. Export before running the factory.
macOS Quick Install

brew install docker node python covers Docker, Node, and Python.

Claude Code CLI: npm install -g @anthropic-ai/claude-code

Setup

  1. Clone the repository 
    git clone https://github.com/your-org/ProjectZeroFactory.git
cd ProjectZeroFactory
export ANTHROPIC_API_KEY=sk-ant-...
  2. Initialize the factory

    Run inside Claude Code once per machine:

    /factory-init

    This spins up Docker containers, runs DB migrations, validates API keys, and writes .factory/ config.

    First Run

    Copy .env.example → .env and fill required vars (DB URL, Temporal address, JIRA key) before running.

  3. Bootstrap your first product 
    /bootstrap-product --name "my-product" --type saas --stack "nextjs,fastapi,postgresql"

    Runs a 13-step Temporal workflow: creates GitHub repo, JIRA project, Confluence space, product scaffold, and auto-initializes the design system (tokens, Framer Motion variants, 6 base components, Storybook).

    Or use the agentic script with a natural-language prompt:

    python scripts/bootstrap-agent.py "bootstrap i-comply for iSourceInnovations"
