# Feature Correctness Checklist

- [ ] Business logic matches specification exactly
- [ ] Calculations verified (with sample data)
- [ ] Data integrity maintained (no orphaned records, no duplicates)
- [ ] API contract honored (request/response matches spec)
- [ ] Database constraints enforced (unique, not null, foreign keys)
- [ ] Concurrency handled (no race conditions)
- [ ] Timezone handling correct (UTC storage, local display)
- [ ] Pagination works correctly (first, last, middle pages)
