# Feature Audit Checklist

- [ ] Code quality: readable, maintainable, no code smells
- [ ] Test coverage: ≥ 80%, meaningful tests (not just line coverage)
- [ ] Security review: no OWASP Top 10 violations
- [ ] Performance: no N+1 queries, no unnecessary re-renders, acceptable response time
- [ ] Accessibility: WCAG 2.1 AA compliant
- [ ] Architecture alignment: follows module boundaries, uses correct patterns
- [ ] Design system compliance: uses tokens and shared components
