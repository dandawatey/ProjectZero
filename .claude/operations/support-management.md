# Support Management

## Workflow
Intake → Categorize → Route → Resolve → Follow Up

## Categories
- Bug report → Engineering (bug management process)
- Feature request → Product Manager (backlog)
- How-to question → Documentation / Knowledge base
- Account issue → Admin team
- Escalation → Engineering lead

## SLA
- First response: 4 hours (business hours)
- Resolution: depends on category and severity
- Follow-up: within 24 hours of resolution
