# Design Principles

## 1. Consistency
Same patterns, same components, same interactions everywhere. Users learn once.

## 2. Accessibility
WCAG 2.1 AA minimum. Keyboard navigable. Screen reader compatible. Color is never the only indicator.

## 3. Responsiveness
Mobile-first. Works at every breakpoint. No horizontal scroll. Touch-friendly.

## 4. Performance
Components render fast. Lazy load where appropriate. No layout shift. Skeleton loading.

## 5. Simplicity
Minimal UI. Show what matters. Progressive disclosure for complexity. No decoration without purpose.

## 6. Feedback
Every action has visible feedback. Loading states. Success confirmation. Error explanation with recovery path.
