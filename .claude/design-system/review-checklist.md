# Design Review Checklist

- [ ] Uses design tokens (no hardcoded colors, sizes, spacing)
- [ ] Responsive at all breakpoints (sm, md, lg, xl)
- [ ] Accessible: keyboard navigable
- [ ] Accessible: screen reader labels (ARIA)
- [ ] Accessible: color contrast ≥ 4.5:1
- [ ] Accessible: focus indicators visible
- [ ] Touch targets ≥ 44x44px on mobile
- [ ] Loading state implemented
- [ ] Error state implemented (with recovery action)
- [ ] Empty state implemented (with guidance)
- [ ] Consistent with existing patterns
- [ ] Storybook stories cover all variants
- [ ] No ad-hoc styling (all from design system)
