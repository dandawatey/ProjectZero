# Usage: Feature Forge

## How to Invoke
Activate this skill when its trigger conditions are met (see triggers.md).

## Process
1. Understand: Read ticket, acceptance criteria, and relevant architecture. 2. Design: Plan implementation approach. 3. Test first: Write failing tests for each acceptance criterion. 4. Implement: Write code to make tests pass. 5. Refactor: Clean up while keeping tests green. 6. Integrate: Ensure feature works with existing code. 7. Validate: Run full test suite, check coverage.

## Expected Output
Completed work following the methodology above, with quality validated against checklist.md.
