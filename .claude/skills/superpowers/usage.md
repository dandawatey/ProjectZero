## Process
1. Shortcuts: Use IDE shortcuts, terminal aliases, git aliases for common operations. 2. Automation: Script repetitive tasks (file generation, boilerplate, test setup). 3. Parallel execution: Use git worktrees for parallel feature work, parallel test suites. 4. Context switching: Checkpoint before switching, use named sessions, quick-resume. 5. Templates: Use code snippets, file templates, generator scripts for consistency.
