# Skill: iOS HIG Design

## Purpose
Design following Apple's Human Interface Guidelines for iOS apps.

## Methodology
1. Navigation: Tab bar for top-level, navigation stack for drill-down, modals for focused tasks. 2. Layout: Safe areas, dynamic type support, standard margins (16pt). 3. Typography: SF Pro system font, Dynamic Type sizes, semantic styles (title, body, caption). 4. Color: System colors that adapt to light/dark mode, tint color for branding. 5. Icons: SF Symbols preferred, consistent weight and size. 6. Gestures: Standard gestures (tap, swipe, long-press), no hidden gestures for primary actions. 7. Accessibility: VoiceOver labels, Dynamic Type, sufficient contrast, motor accessibility.
