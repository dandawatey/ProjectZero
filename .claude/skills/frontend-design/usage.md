# Usage: Frontend Design

## How to Invoke
Activate when trigger conditions are met.

## Process
1. Component hierarchy: Map pages to component trees, identify shared vs page-specific. 2. State management: Choose approach (React Query for server state, Zustand/Context for client state). 3. Routing: Define routes, protected routes, layout nesting. 4. API integration: Type-safe API client from contracts, error handling, loading states. 5. Performance: Code splitting, lazy loading, memoization where needed.
