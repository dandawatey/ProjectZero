# Skill: UI/UX Pro Max

## Purpose
Comprehensive UI/UX process from research to polished implementation.

## Methodology
1. Research: Understand users (personas, jobs-to-be-done, pain points). 2. Wireframe: Low-fidelity layouts showing information architecture and flow. 3. Prototype: Interactive prototype for key flows (Figma, code prototype). 4. Test: User testing with 5 users — identify usability issues. 5. Iterate: Fix top issues, re-test critical flows. 6. Polish: Final visual design using design tokens, micro-interactions, transitions.
