# Usage: RAG Architect

## How to Invoke
Activate when trigger conditions are met (see triggers.md).

## Process
1. Define use case: What questions need answering? What data sources? 2. Chunking strategy: Select chunk size, overlap, method (semantic, fixed, recursive). 3. Embedding model: Choose model (OpenAI, Cohere, open-source) based on domain and cost. 4. Index design: Vector store selection, metadata filtering, hybrid search. 5. Retrieval: Top-k, reranking, query expansion. 6. Generation: Prompt template, context window management, citation. 7. Evaluation: Relevance scoring, answer quality, hallucination detection.
