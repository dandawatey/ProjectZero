# Skill: Design Sprint

## Purpose
Google Ventures design sprint: solve big problems in 5 days (compressed for AI execution).

## Methodology
Day 1 - Map: Define the problem, map the user journey, pick a target area.
Day 2 - Sketch: Generate solutions individually, present and discuss, vote on best ideas.
Day 3 - Decide: Choose winning solution, create storyboard for prototype.
Day 4 - Prototype: Build realistic prototype (just enough to test, not production).
Day 5 - Test: Test with 5 target users, capture feedback, decide next steps.
