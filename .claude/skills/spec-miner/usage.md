# Usage: Spec Miner

## How to Invoke
Activate this skill when its trigger conditions are met (see triggers.md).

## Process
1. Parse document: Read full BMAD/PRD. 2. Identify sections: Map to BMAD template sections. 3. Extract requirements: Pull out specific, testable requirements. 4. Structure as specs: Format into epics, stories, acceptance criteria. 5. Identify gaps: Flag missing information. 6. Validate: Cross-reference requirements for consistency.

## Expected Output
Completed work following the methodology above, with quality validated against checklist.md.
