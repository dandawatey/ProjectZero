# Usage: Debug Skill

## How to Invoke
Activate this skill when its trigger conditions are met (see triggers.md).

## Process
1. Reproduce: Create reliable reproduction steps. 2. Isolate: Narrow to smallest failing case. 3. Diagnose: Read error messages, check logs, add targeted logging. 4. Hypothesize: Form theory about root cause. 5. Fix: Implement targeted fix. 6. Verify: Confirm fix resolves issue and doesn't break other tests. 7. Document: Record the pattern in debug-patterns.md.

## Expected Output
Completed work following the methodology above, with quality validated against checklist.md.
