# Usage: Hooked UX

## How to Invoke
Activate when trigger conditions are met (see triggers.md).

## Process
1. Trigger: Define external triggers (email, notification) and internal triggers (emotion, habit). 2. Action: Make the desired action as simple as possible (Fogg: motivation + ability + trigger). 3. Variable Reward: Provide rewards that vary — social (connection), hunt (information), self (mastery). 4. Investment: User invests something (data, effort, content) that makes product more valuable next time.
