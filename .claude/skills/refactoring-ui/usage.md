# Usage: Refactoring UI

## How to Invoke
Activate when trigger conditions are met (see triggers.md).

## Process
1. Identify patterns: Find repeated UI patterns across the app. 2. Map to components: Match patterns to existing design system components. 3. Extract: Create shared components for patterns without matches. 4. Apply tokens: Replace hardcoded values with design tokens. 5. Validate: Visual regression, accessibility, responsiveness.
