# ProjectZero Design System

**Glassmorphism Enterprise UX** — PRJ0-84

## Design Direction
Frosted-glass surfaces · Subtle transparency · Soft layered elevation · Dark & light mode · WCAG accessible

## Stack
React · TypeScript · Tailwind CSS · Framer Motion · Storybook

## Rules
- Build UI by composing from this library FIRST
- Never create one-off components without adding them here
- All components: token-driven, accessible, dark/light aware

## Quick Import
```tsx
import { Button, Card, GlassPanel, Modal, Badge } from '../components/ui';
```
